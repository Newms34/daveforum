About

This site is the Brethren [PAIN] Guild Wars 2 guild website (henceforth just known as "the site")! If you're reading this, you're likely already a member of the Brethren guild. Welcome!

Accounts

Before using the site (and actually, viewing this page), you’re gonna need to create a account. You may note that creating an account involves just creating a username and password, and not any other information. This is because while that other info (i.e., your characters, preferred game mode, etc.) is cool, I don’t want to force you to include it before you sign up.


Confirming

Once you’ve created your account, you’ll need it to be confirmed. This is so that while anyone can create an account, only users confirmed by Brethren site moderators can actually view the site. Trolls r bad, k?

Again, to clarify, if you're seeing this page, you're already confirmed.

General

This help section applies, generally speaking, to stuff posted in the  Chat,  Forum, and   Blog Editor sections.

While you can just post regular old boring text, you can also make it more interesting with my special flavor of something called Markdown.

Healy's Markdown

Restrictions

There are a few technical restrictions with what you can post

  • You cannot post custom HTML. Any HTML will not be parsed, and will be displayed as the raw HTML "code".
  • Github, etc. "mentions" do not work. Nor should they, if you're wondering.
